### Installation

#### Use it without installing

``` sh
curl -s https://raw.githubusercontent.com/yrwq/yafetch/main/yafetch | sh
```

#### Use it locally

- clone the repo  

``` sh
git clone https://github.com/yrwq/yafetch && cd yafetch
```

- make it executable

``` sh
chmod +x yafetch
```

- copy it to your $PATH

``` sh
cp yafetch ~/.local/bin/
```

``` sh
yafetch
```
